Siga bem

Nosso trabalho é um chatbot que possa coletar e armazenar informações claras, de maneira que quando o cliente for transferido ao atendente, o atendente tenha todas as informações necessárias para escolher a melhor opção para suprir as necessidades do cliente.

Interface

Também faremos um sistema com uma interface amigável para que o atendente possa ter facilidade ao concluir a chamada, se ela não for resolvida antes de maneira automatizada.

Chat Bot

O chatbot validará se o cliente possui o cpf cadastrado no sistema, e analisará as informações que estão cadastradas, caso falte alguma informação, ele vai solicitar a informação ao cliente, após coleta de todas as informações enviará uma mensagem de confirmação ao usuário com as informações para que o cliente confirme que elas estão certas ou não.

Atendente

Após a confirmação do cliente, o sistema perguntará qual o tipo de incidente que o cliente está passando, após respondido direcionará para um atendente com todas as informações obtidas.

Melhoria

Possível melhoria, sistema já contactar o serviço pertinente ao problema do cliente assim não havendo necessidade de direcionar para atendente ao final do processo
